Difference Between map() and forEach() in JavaScript

Both map() and forEach() are used to iterate over arrays

Map

  • It is used to transform each item in the array.
  • It returns A new array with transformed values.
  • It Can be chained with other methods like filter(), reduce(), etc.
  • It does not mutate the original Array

forEach

  • It is used to perform a side effect (like console logging, updating DOM, etc.) on each item.
  • It returns undefined
  • It cannot be chained â€” because it returns nothing.
  • It does not mutate the original Array
