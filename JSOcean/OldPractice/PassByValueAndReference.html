What is difference between Pass By Value and Pass By Reference ?

Pass by Value

  • When a variable is passed by value, a copy of the variable's value is passed to the function.
  • So, changes made inside the function do not affect the original value
  • Applies to: Primitive types - String, Number, Boolean, undefined, null, Symbol, BigInt

Pass by Reference

  • When a variable is passed by reference, the reference (memory address) is passed to the function
  • So, changes made inside the function affect the original object/array.
  • Applies to: Non-primitive types - Object, Array, Function
