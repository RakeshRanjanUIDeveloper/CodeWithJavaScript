For...in

  • Iterates over enumerable properties (keys) of an object
  • Returns keys/indexes (as strings)
  • Best for objects
  • Objects, Arrays (not recommended for arrays)

for...of

  • Iterates over iterable values (like arrays, strings, maps)
  • Returns values directly
  • Best for arrays, strings, maps, etc.
  • Arrays, Strings, Sets, Maps, NodeLists, etc.
