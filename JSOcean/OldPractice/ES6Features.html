Feature Description Example
let / const Block-scoped variable declarations 
let x = 10;
const y = 20;
Arrow Functions Short function syntax with lexical `this` 
const greet = (name) => `Hello, ${name}`;
Template Literals Multi-line and embedded expressions 
const msg = `Hello
World!`;
Default Parameters Set default values for function parameters 
function multiply(a, b = 2) {
  return a * b;
}
Destructuring Extract values from objects/arrays 
const { name, age } = { name: "Rakesh", age: 30 };
Spread & Rest Expand or collect values 
const arr = [1, 2, 3];
const newArr = [...arr, 4];
Enhanced Object Literals Simplified object properties and methods 
const age = 25;
const user = { name: "Rakesh", age };
Promises Handle async operations 
fetch(url).then(res => res.json());
Classes Simplified syntax for OOP 
class Person {
  constructor(name) {
    this.name = name;
  }
}
Modules Use `import` / `export` to share code 
// file.js
export const msg = "Hello";

// main.js
import { msg } from './file.js';
for...of Iterate over iterable objects 
for (let num of [1, 2, 3]) {
  console.log(num);
}
Map & Set New data structures 
const ids = new Set([1, 2, 2, 3]);
Symbol Unique and immutable identifiers 
const id = Symbol('id');
Generators Function that can pause with `yield` 
function* gen() {
  yield 1;
  yield 2;
}
