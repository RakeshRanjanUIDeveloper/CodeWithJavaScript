Object Basics Operations

  • Check if an Object is empty
    • Object.keys(obj) returns an array of the object
  • Merge Two Objects
    • The spread syntax (...) copies all enumerable properties from obj1 and obj2 into a new object.
    • If both objects have the same key, the value from the second object (obj2) will overwrite the one from obj1.
  • Clone an Object (shallow copy)
    • Only the top-level properties are copied.
    • Use shallow copy only when your object does not have nested objects
    • Using Spread Operator { ...obj }
    • Using Object.assign()
    • If we change a primitive value in the shallow copy, it does not affect the original object.
    • If we change a nested object or array, it does affect the original — because only the reference is copied.
  • Deep Clone an Object (Deep Copy)
    • Using structuredClone()
    • Using JSON.parse(JSON.stringify(obj))
    • A deep copy creates a completely new and independent copy of every level of the object — including all nested objects and arrays.
    • If we change a nested object or array, it does not affect the original — because a completely new copy of the nested structure is created.
