What are the different position values in CSS? What is the default position?

  • position determines how an element is placed in the document and interacts with other elements.
  • static (default): Element follows the normal document flow.
  • relative: Positioned relative to its original position. Still takes up space in layout.
  • absolute: Positioned relative to the nearest positioned ancestor. Removed from normal flow.
  • fixed: Positioned relative to the viewport. Stays fixed on scroll.
  • sticky: Behaves like relative until it reaches a scroll threshold, then becomes fixed.
