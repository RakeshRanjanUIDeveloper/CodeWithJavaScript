charAt() Method in JavaScript

The charAt() method returns the character at a specified index in a string.

Syntax

string.charAt(index)

Parameters

  • index: A number (0-based) that specifies the position of the character to return.
  • If the index is out of range, it returns an empty string "".
  • If the index passed to the charAt() method is negative, it will return an empty string ("").

Common Use Cases

  1. Check if a string starts with a specific character
  2. Count occurrences of a character in a string
  3. Filter out unwanted characters
  4. Validate inputs character by character
