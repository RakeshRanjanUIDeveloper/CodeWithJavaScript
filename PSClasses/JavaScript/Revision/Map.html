A Map is a collection of key-value pairs where:

  • Keys can be of any type (objects, functions, primitives).
  • Insertion order is preserved
  • Map is iterable, while regular objects are not directly iterable.

Syntax - const myMap = new Map();

You can also initialize a map with values

  • set(key, value) Adds or updates an element
  • get(key) Returns the value for the given key
  • has(key) Checks if the key exists
  • delete(key) Deletes the key-value pair
  • clear() Removes all elements
  • size Returns the number of elements in the map
  • keys() Returns an iterator for keys
  • values() Returns an iterator for values
  • entries() Returns an iterator for key-value pairs
  • forEach() Executes a function for each key-value pair
