What Does "Isomorphic" Mean in Strings?

  1. Two strings are isomorphic if the characters in one string can be replaced to get the second string, with the order preserved
  2. Each character from the first string maps to one and only one character in the second string.

Rules

  1. Each character must map to another character uniquely
  2. The mapping must be one-to-one.
  3. The structure or pattern of the characters must match.
