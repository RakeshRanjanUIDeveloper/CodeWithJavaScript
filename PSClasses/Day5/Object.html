JavaScript Object Methods Explained

hasOwnProperty

  • Checks whether the object has the specified property as its own (not inherited from the prototype chain).
  • object.hasOwnProperty(property)

Object.keys()

  • Returns an array of the object's own enumerable property names (keys).

Object.values()

  • Returns an array of the values of the objectâ€™s own enumerable properties.

Object.entries()

  • Returns an array of key-value pairs, where each pair is an array [key, value].

Object.fromEntries()

  • The reverse of Object.entries().
  • It converts an array of key-value pairs back into an object.
Method Purpose Output Example
hasOwnProperty Check if object owns a property true or false
Object.keys() Get array of keys ["name", "age"]
Object.values() Get array of values ["Rakesh", 30]
Object.entries() Get array of [key, value] pairs [["name", "Rakesh"], ["age", 30]]
Object.fromEntries() Convert entries back to object { name: "Rakesh", age: 30 }
