It is a built-in string and array method used to find the position of a specified value within a string or an array.

  1. For Strings: String.prototype.indexOf()
  2. For Arrays: Array.prototype.indexOf()

Syntax:

  1. str.indexOf(searchValue, fromIndex)
  2. searchValue (required): The string to search for.
  3. fromIndex (optional): The index to start the search from. Defaults to 0.

Returns:

  1. The index (0-based) of the first occurrence of the specified substring.
  2. Returns -1 if the value is not found.
