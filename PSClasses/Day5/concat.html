concat() Method in JavaScript

The concat() method in JavaScript is used primarily with arrays and strings.

It’s a non-mutating method, meaning it doesn’t change the original array or string—it returns a new one.

Syntax

let newArray = array1.concat(array2, array3, ...);

What it Does

  • Combines two or more arrays.
  • Returns a new array, leaving the originals untouched.

concat() with Strings

let newString = string1.concat(string2, string3, ...);
  • Joins two or more strings into one.
  • Returns a new string.
