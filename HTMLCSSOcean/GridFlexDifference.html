What is the difference between CSS Grid and Flexbox? When should you use each?

  • Grid is used for 2D layout (both rows and columns). Flexbox is used for 1D layout (either row or column).
  • Grid is great for building complex layouts. Flexbox is better for simple, linear UI elements.
  • Grid allows overlapping content (using grid lines). Flexbox does not support overlap by default.
  • Grid allows control of row and column separately. Flexbox aligns content in a single axis (row or column).
  • Grid parent: `display: grid` / `inline-grid`. Flexbox parent: `display: flex` / `inline-flex`.
